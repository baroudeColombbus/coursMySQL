Accueil requete insert update & DELETE

Les requêtes de modification avec UPDATE

La requête UPDATE

  • Augmenter le salaire d'un employé : UPDATE SET salaire = 1871 WHERE id_employes = 990;
    Il est recommendé de modifier un enregistrement en ciblant son id, l'id étant unique

La requête DELETE

  • Supprimer un enregistrer; DELETE FROM employes WHERE id_employes = 388;
  • Supprimer plusieurs enregestritements : DELETE FROM employes WHERE service = 'informatique';
