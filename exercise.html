cours SQL
Accueil requete insert update & DELETE
  1. Afficher la profession de l'employé 547 : SELECT service FROM employes WHERE id_employes=547;
  2. Afficher la date d'embauche d'Amandine : SELECT prenom, date_embauche FROM employes WHERE prenom='Amandine';
  3. Afficher le nombre de commerciaux : SELECT COUNT(*) FROM employes WHERE service ='commercial';
  4. Afficher le coût des commerciaux sur 1 année : SELECT COUNT(*) AS 'nbr_commerciaux' FROM employes WHERE service ='commercial'; avec alias
  5. Afficher le salaire moyen par service : SELECT AVG(salaire) AS 'salaire_moyen_par_service' FROM employes GROUP BY service; on peut aussi l'amélioré comme ici :
    SELECT service,ROUND ( AVG(salaire)) AS 'salaire_moyen_par_service' FROM employes GROUP BY service ORDER BY salaire DESC;
    SELECT service, AVG(salaire) AS 'cout_moyen' FROM employes GROUP BY service; ou encore
    grouper les serives par ordre alphabétique SELECT service,ROUND ( AVG(salaire)) AS 'salaire_moyen_par_service' FROM employes GROUP BY service ASC;
  6. Afficher le nombre de recrutements sur l'année 2010 : SELECT COUNT(date_embauche) AS nombre_d_emauche FROM employes WHERE date_embauche BETWEEN '2010-01-01' AND '2010-12-31';
    SELECT COUNT(*) as 'nb de recrutement' FROM employes WHERE date_embauche >= '2010-01-01' AND date_embauche <= '2010-12-31'; ou encore
    SELECT COUNT(*) AS 'nbr de recrutement 2010' FROM employes WHERE date_embauche LIKE '2010%';
  7. Augmenter le salaire pour chaque employé de 100 € : UPDATE employes SET salaire=salaire+100;
  8. Afficher le nombre de services (différents) : SELECT DISTINCT service FROM employes;
  9. Afficher les informations de l'employé du service commercial gagnant le salaire le plus élevé : SELECT * FROM employes WHERE service ='commercial' AND salaire=(SELECT MAX(salaire) FROM employes WHERE service='commercial');
  10. Afficher l'employé ayant été embauché en dernier : SELECT date_embauche,nom, prenom, salaire,service, sexe FROM employes WHERE date_embauche = (SELECT MAX(date_embauche) FROM employes);
