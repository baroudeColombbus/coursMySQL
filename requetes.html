Le mot-clef LIKE

Afficher les employés dont le prénom commence par s

Avec la présence des signes pourcentages avant et après la lettre ou le mot recherché, nous demandons au systmème de trouver des enregestritements contenant l'expression recherchée.

  • SELECT nom, prenom FROM employes WHERE prenom LIKE 's%';
  • SELECT * FROM employes WHERE prenom LIKE %s
  • SELECT * FROM employes WHERE prenom LIKE s%c

6- Le classement avec ORDER BY

  • Les employés triés par nom de famille et par ordre alaphabétique SELECT nom,prenom, salaire FROM employes ORDER BY nom ASC;
  • Avec ou sans ASC la requête est la même par default SQL tri du plus petit au plus grand
  • En ordre descendant ou décroissant DESC
  • Trier sur plusieurs colonnes : SELECT service,nom,prenom, salaire FROM employes ORDER BY service ASC, salaire ASC;

7- Limiter les résultats avec les mot-clef LIMIT

  • Afficher les employés en les classant par salaire (du + grand au plus petit)et en les affichant par 3 SELECT * FROM employes ORDER BY salaire DESC LIMIT 0,3;
  • Le premier chiffre précise l'enregistrement de départ(0 définit le 1er enregistrement en informatique) le 2nd chiffre nous indique le nombre de résultats que l'on souhaite obtenir
  • Autre ex : SELECT, nom,prenom, service, salaire FROM employes ORDER BY salaire DESC LIMIT 6,3
  • Le mot-clef ou la clause LIMIT est utile pour faire une pagination web

8- Réaliser les calculs

  • SELECT nom, prenom, salaire,salaire*12 as salaire_annuel FROM employes ORDER BY salaire DESC;
  • Le mot-clef AS permet de définir un alias

Calculer une somme avec SUM(), et une moyenne avec AVG()

  • SUM() faire une somme, addition : SELECT SUM(salaire*12) FROM employes;
  • AVG() faire la moyenne annuelle (average) : SELECT AVG(salaire*12) FROM employes;

Arrondir avec ROUND

  • Arrondir un calcul : SELECT ROUND( AVG(salaire*12))FROM employes;
  • SELECT ROUND( AVG(salaire))AS salaire_mensuel_arrondie FROM employes;
  • Combien il y a de femme : SELECT COUNT(*) FROM employes WHERE sexe='m';

Isoler une valeur minimum/maximum avec MIN() ou MAX()

  • Trouver le salaire minimum : SELECT MIN(salaire) FROM employes;
  • Trouver le salaire maximum : SELECT MAX(salaire) FROM employes;
  • Par contre cette requête ne fonctionnera pas : SELECT prenom, MIN(salaire) FROM employes;, pour trouver le salaire le plus bas et la personne qui gagne ce salaire il faut écrire SELECT prenom, salaire FROM employes WHERE salaire = (SELECT MIN(salaire) FROM employes); : Le système va d'abord d'exécuter la requête entre parenthèse pour connaître le résultat et l'utiliser comme source pour l'autre requête
  • Trouver qui gagne le plus : SELECT nom, prenom FROM employes WHERE salaire = (SELECT Max(salaire) FROM employes);
  • >Trouver qui gagne moins : SELECT nom, prenom FROM employes WHERE salaire = (SELECT MIN(salaire) FROM employes); on appelle cela une requête imbriqué
  • Trouver les employes qui gagne SELECT nom, prenom,salaire FROM employes WHERE salaire = 1390;

    • Condition IN avec plusieurs valeurs

    Conditions multiples, différent champs et valeurs

    • Les employes qui travaillent au service commercial et qui gagne un salaire inférieur ou égal à 2000 euros
      SELECT nom, prenom, salaire, service FROM employes WHERE service = 'commercial' AND salaire <= 2000

    Ordre de priorité sur les conditions AND et OR

    • Lorsqu'il y a AND et OR au sein de la même requête, le systmème lit la condition de la droite vers la gauche
    • SELECT nom, prenom, salaire FROM employes WHERE salaire IN (1900, 2300) AND service='production';
    • Pour garantir l'ordre naturel de lecture (de la gauche vers la droite) on utilisera les paramthèse
      SELECT prenom, nom, service, salaire FROM employes WHERE service='production' AND (salaire=1900 OR salaire=2300);

    Les regroupement avec GROUP BY

    • Si on veut connaître le nombre d'employés par service : SELECT service, COUNT(*) AS nombre_employe FROM employes GROUP BY service

    Utilisation de mot mot-clef HAVING, ayant

    • On impose une condition COUNT et HAVING SELECT service, COUNT(*) AS nombre_employe FROM employes GROUP BY service HAVING COUNT(*);
